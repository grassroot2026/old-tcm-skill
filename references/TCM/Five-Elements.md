| 五行 | 五脏 | 五色 | 五音 | 五季 | 五时 | 五情 |

## 🔮 五行相生 (生我/我生)

```
木 → 火 → 土 → 金 → 水 → 木 (循环)
生：木生火，火生土，土生金，金生水，水生木
```

**生我者**为母，**我生者**为子

### 病理关系
- **母病及子**: 肝木不舒→心火不足
- **子病及母**: 肾水不足→肝木虚亢

## 💥 五行相克 (克我/我克)

```
木 → 土 → 水 → 火 → 金 → 木 (循环)
克：木克土，土克水，水克火，火克金，金克木
```

**克我者**为仇，**我克者**为臣

### 病理关系
- **乘 (过度克制)**: 肝木过盛→过度克制脾土
- **侮 (反向克制)**: 土弱木盛→反克脾土

## 🌕 临床应用

### 治疗原则

### 虚则补其母
- **肝虚**: 补心火 (母病及子)
- **肾虚**: 补肝木 (水生木)

### 实则泻其子
- **肝实**: 泻心火 (泻子)
- **脾实**: 泻胃土 (泻子)
