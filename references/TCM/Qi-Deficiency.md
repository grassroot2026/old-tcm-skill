# 气虚辨证：五脏虚证分析

老中医思维：气虚证的系统辨证

## ✨ 气的功能

### 生理作用
- **推动作用**: 脏腑功能、血液运行
- **温养作用**: 维持体温、防止寒凝
- **固摄作用**: 控制体液、防止出血
- **防御作用**: 抵御外邪、抗御病邪
- **气化作用**: 物质转化、新陈代谢

### 气机运动
- **升**: 脾气上升、肝气升发
- **降**: 肺气下降、胃气下行
- **出**: 呼气、排便
- **入**: 吸气、纳谷

## 🌫️ 气虚证的识别

### 共见症状
- 神疲乏力 (核心症状)
- 少气懒言
- 自汗出
- 动则加重
- 舌淡苔白
- 脉弱无力

### 五脏气虚特征

#### 肺气虚
```
核心症状：气短懒言
伴随症状：声音低微、易感冒、咳喘无力、自汗
舌脉：舌淡，脉弱
病机：肺主气功能减弱
```

#### 心气虚
```
核心症状：心悸怔忡
伴随症状：胸闷、气短、神疲、自汗
舌脉：舌淡白，脉弱或结代
病机：心主血脉功能不足
```

#### 脾气虚
```
核心症状：食少便溏
伴随症状：腹胀纳呆、肢体倦怠、消瘦
舌脉：舌淡胖，苔白，脉缓
病机：脾主运化功能减退
```

#### 胃气虚
```
核心症状：腹胀胃痞
伴随症状：脘腹隐痛、喜按、食后加重
舌脉：舌淡，脉弱
病机：胃主受纳腐熟失职
```

#### 肾气虚
```
核心症状：腰酸膝软
伴随症状：小便频数、阳痿早泄、遗精滑泄
舌脉：舌淡，脉沉弱
病机：肾藏精主生殖功能减退
```

## 🩺 辨证要点

### 鉴别要点

| 气虚类型 | 特征症状 | 鉴别关键 |
|---------|---------|---------|
| 肺气虚 | 气短懒言、易感冒 | 表卫不固 |
| 心气虚 | 心悸怔忡 | 胸闷气短 |
| 脾气虚 | 食少便溏 | 运化失职 |
| 胃气虚 | 脘痞腹胀 | 受纳腐熟 |
| 肾气虚 | 腰酸膝软 | 生殖泌尿 |

## 💡 治疗思路

```
气虚证 → 辨证定位 (何脏气虚) 
    ↓
治则：补益其脏
    ↓
补益脏腑之气
    ↓
佐以调和气机
```

### 常用方剂
- **肺气虚**: 玉屏风散、补肺汤
- **心气虚**: 炙甘草汤、保元汤
- **脾气虚**: 四君子汤、六君子汤
- **胃气虚**: 异功散、参苓白术散
- **肾气虚**: 金匮肾气丸、右归丸

## ⚠️ 注意事项

```
1. 气虚易兼夹：气虚夹痰、气虚夹瘀
2. 虚实转化：久病气虚可化热成郁热
3. 母子关系：五脏气虚相互关联
```
